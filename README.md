# SADY-OPILLIA (Online Garden Market)

SADY-OPILLIA is a full-stack JavaScript e-commerce platform specialized in selling saplings, young fruit trees, and plants. This project showcases a complete client-server architecture.

## Features
* **Product Catalog:** Browse various categories of plants and trees.
* **Shopping Cart:** Add, remove, and manage items before checkout.
* **Full-Stack JS:** REST API communication between the frontend client and the backend server.

## Technologies
* **Backend:** Node.js, ExpressJS
* **Frontend:** NextJS
* **Database:** MongoDB
